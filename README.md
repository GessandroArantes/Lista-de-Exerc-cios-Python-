# 🐍 Curso SENAI - Primeiros Passos em Python

Este repositório contém os exercícios e projetos desenvolvidos durante o curso no **SENAI**. O foco principal é o aprendizado da lógica de programação utilizando a linguagem **Python**.

## 🚀 Conteúdo do Repositório

Os arquivos aqui presentes cobrem diversos fundamentos da programação, incluindo:

* **Estruturas Condicionais:** Exemplos práticos com `if`, `elif` e `else` (ex: `aula_if.py`).
* **Laços de Repetição:** Exercícios utilizando `while` e `for` (ex: `laco_repeticao.py`).
* **Manipulação de Dados:** Scripts para entrada e saída de dados (ex: `input_python.py`).
* **Lógica de Programação:** Diversos desafios lógicos e matemáticos aplicados.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem Principal:** Python 100%
* **Plataforma de Desenvolvimento:** Scripts focados em execução via terminal.

## 📂 Como visualizar os códigos
Cada arquivo `.py` representa um exercício específico. Você pode abrir qualquer um deles diretamente aqui no GitHub para visualizar a lógica aplicada em cada aula do SENAI.
